# ProMed Technologies - IT Support & System Guide

## Intranet Access

### Logging In
1. Open your web browser (Chrome or Edge recommended)
2. Navigate to the ProMed Intranet URL
3. Enter your email address and password
4. Click Login

### Password Reset
If you forgot your password or need it reset:
- Contact an administrator or IT support
- An admin can reset your password via User Management
- Welly can reset passwords for admins: "Reset password for [name]"
- After reset, you will receive a temporary password to log in with

### Account Issues
- If your account is deactivated, contact an administrator
- Admins can activate/deactivate accounts via User Management
- Module permissions control which sections you can access

## Support Ticket System

### Submitting an IT Support Ticket
1. Go to Support Tickets from the main navigation
2. Click "New Ticket"
3. Enter a descriptive title
4. Describe the issue in detail
5. Select priority: Low, Medium, High, or Critical
6. Submit the ticket

### Tracking Your Ticket
- View all your open tickets on the Support Tickets page
- Check ticket status: Open, In Progress, Pending, Resolved, Closed
- Add comments to provide additional information
- Get notified when your ticket is updated

### Ticket Statistics
- View open ticket count and resolution statistics
- Filter tickets by status (open, closed)
- Admins can assign tickets to specific staff members

### Using Welly for Support
Ask Welly: "Create a support ticket for [describe your issue]"
Welly will create the ticket automatically on your behalf.

## Module Access & Permissions

### Available Modules
The intranet has the following modules that require permission:
- **Sales**: Sales dashboard, imports, CRM
- **Logistics**: Full logistics operations
- **Tenders**: Tender management and compliance
- **Project Management**: Kanban boards and projects
- **Human Resource**: Employee directory, attendance, leave
- **Stock Management**: Inventory and warehouse management
- **Support Tickets**: IT support ticketing
- **Messaging**: Internal messaging system
- **AI**: Access to Welly AI assistant
- **PBX**: Phone system and call history
- **Calendar**: Meetings and events
- **Settings**: System configuration

### Requesting Module Access
If you cannot access a module you need:
1. Submit a support ticket requesting access
2. Your manager or admin will review and grant permissions
3. Permissions are updated in your user profile

## User Roles

### Role Types
- **Standard User**: Basic access to assigned modules
- **Manager**: Team management, approvals, reports
- **Admin**: Full system configuration and user management
- **Super Admin**: Complete system control including database operations

## Troubleshooting

### Page Not Loading
1. Check your internet connection
2. Clear your browser cache (Ctrl+Shift+Delete)
3. Try a different browser
4. Check if the intranet server is reachable
5. Submit a support ticket if the issue persists

### Cannot Log In
1. Verify your email address is correct
2. Check that Caps Lock is off
3. Try resetting your password through IT
4. Ensure your account is active (contact admin)

### Data Not Displaying
1. Refresh the page (F5 or Ctrl+R)
2. Check if any filters are applied
3. Verify you have permission to view that module
4. Clear browser cache and try again
5. Report the issue via a support ticket

### Real-Time Features Not Working
The intranet uses SignalR for real-time updates (attendance, messaging, Kanban boards, collaborative documents). If live updates stop:
1. Refresh the page
2. Check your internet connection
3. The system will attempt to reconnect automatically

## Welly AI Assistant (Quick IT Help)

Welly can help with common IT tasks:
- "Reset password for [name]" (admin only)
- "Create a support ticket for [issue]"
- "What modules do I have access to?"
- "How do I [perform a task]?"

Welly can also search the web for technical answers using the built-in web search feature.

## Data Backups

### Database Backups
- The system supports automatic and manual database backups
- Admins can trigger backups via the Database settings page
- Backups are stored as SQL Server backup files
- Backup history is available in the admin panel
- Welly can initiate backups for admins: "Backup the database"

### Data Export
- Database tables can be exported individually as JSON
- Full database export is available for admins
- Audit logs can be exported for compliance

## Audit Logs

All significant system actions are logged:
- Login/logout events
- Data changes (creates, updates, deletes)
- Permission changes
- System administration actions
- Logs include timestamp, user, action, and details

## Contact IT Support

For urgent IT issues:
- Submit a Critical priority support ticket
- Use the internal messaging system to contact IT department
- Ask Welly for immediate guidance on common issues
