# ProMed Technologies - Stock & Inventory Knowledge Base

## Stock on Hand (SOH)

### Overview
Stock on Hand tracks inventory levels across all warehouse locations. Data is imported from SYSPRO ERP via Excel files.

### SOH Import Process
1. Export SOH report from SYSPRO as Excel (CSV/XLSX)
2. Go to Stock Management module
3. Click "Import SOH"
4. Upload the Excel file
5. Review import preview (check for issues)
6. Commit to save stock levels to the system
7. View import issues and export as CSV if needed

### SOH Import Features
- Preview before committing (review data before saving)
- Issue detection (missing fields, invalid data)
- Export issues as CSV for correction
- Delete incomplete imports
- View import details and line items

### SOH Fields
- **Item Code**: Unique product identifier from SYSPRO
- **Description**: Product name/description
- **Quantity**: Units in stock
- **Unit of Measure**: Each, Box, Pallet, etc.
- **Bin Location**: Physical location in warehouse
- **Total Cost**: Value of stock

## 3D Warehouse Visualization

### What is the 3D View?
The Warehouse 3D page provides a visual representation of warehouse layout:
- See inventory positioned in 3D space
- View commodity locations within buildings
- Drag items to update positions
- Visualize stock distribution

### Accessing 3D View
- Go to Stock Management or Logistics > Warehouses
- Select a warehouse
- Click "3D View" to open the visualization
- Items can be positioned in 3D space with X, Y, Z coordinates

## Stock Transfers

### Transfer Process
1. Go to Stock Management or Logistics > Warehouses
2. Click "Stock Transfers"
3. Create new transfer
4. Select source warehouse/building
5. Select destination warehouse/building
6. Add items and quantities
7. Submit transfer
8. Update transfer status (dispatched, received)

### Transfer Types
- **Warehouse to Warehouse**: Transfer between GP, KZN, CPT, PE
- **Building to Building**: Transfer within the same warehouse

### Transfer Statuses
- Pending
- In Transit / Dispatched
- Received / Completed

## Stock Movements

### Movement Tracking
All stock changes are recorded as movements:
- Transfers in and out
- Adjustments (increase/decrease)
- View movement history with dates, quantities, and types

### Movement Types
- **Transfer**: Stock moved between locations
- **Adjustment**: Manual quantity correction
- **In**: Stock received
- **Out**: Stock dispatched

## Stock Adjustments

### When to Adjust
- Physical stock count reveals discrepancies
- Damaged goods need to be written off
- Expired products removed from inventory
- System corrections after audits

### Adjustment Process
1. Go to Inventory section
2. Select item to adjust
3. Create stock adjustment
4. Enter reason and new quantity
5. Submit adjustment (recorded in movements)

A: Go to Stock Transfers > Create Transfer > Select source and destination > Add items > Submit

A: Create a stock adjustment with reason "Damage" for the affected items
