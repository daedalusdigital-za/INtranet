# ProMed Technologies - System Features & Navigation

## Application Overview

The ProMed Intranet is a comprehensive business management system built for ProMed Technologies. It covers:
- Employee management and HR (attendance, leave, profiles)
- Project management (Kanban boards)
- Logistics and delivery operations (tripsheets, loads, fleet, TFN)
- Stock and inventory management (warehouses, SOH imports, 3D view)
- Sales and CRM (customer management, sales imports)
- Tender management and compliance
- Support ticketing
- Internal messaging and collaborative documents
- Company announcements
- Meetings and calendar
- PBX phone system integration (3CX)
- AI assistant (Welly)
- Document management
- Database administration
- Audit logging

## Welly AI Assistant

### What is Welly?
Welly is the AI assistant built into the ProMed Intranet. Welly appears as a chat bubble in the corner of every page. Welly is powered by a local AI model (Gemma 4 26B MoE) running on the server.

### What Can Welly Do?
- Answer questions about company data and employees
- Look up employee information, extensions, and departments
- Check attendance and leave status
- Query logistics data (loads, tripsheets, drivers, vehicles)
- Find stock information across warehouses
- Search the knowledge base for company policies and procedures
- Search the web for external information
- Create support tickets
- Schedule meetings
- Assign tasks (todos)
- Send emails
- Create announcements
- Track vehicles
- Get fleet status
- Update customer information
- Reset passwords (admin)
- Get system overview statistics
- Backup the database (admin)
- Analyze uploaded documents (PDF, Excel, CSV)

### How to Use Welly
1. Click the chat icon (bubble) in the bottom corner
2. Type your question or request
3. Welly responds with relevant information or performs actions
4. Welly remembers context within a conversation session

### Example Questions for Welly
- "Who is on leave today?"
- "What's the status of load LD-000123?"
- "How many active tripsheets are there?"
- "What stock do we have at KZN warehouse?"
- "Who manages the Cape Town warehouse?"
- "Show me today's attendance"
- "What's John's extension number?"
- "Create a support ticket for printer not working"
- "Schedule a meeting with Sarah tomorrow at 2pm"
- "Assign a task to Mike to review the report by Friday"
- "Search the web for latest medical device regulations"
- "What is the leave policy?"
- "Track vehicle ABC123GP"
- "Get fleet status"
- "Backup the database"

## User Roles

### Standard User
- View own data and assigned modules
- Submit requests (leave, tickets, tasks)
- Use Welly AI assistant
- Access allowed modules

### Manager
- Everything a standard user can do
- View team data and attendance
- Approve/reject leave requests and sleep outs
- Run departmental reports
- Create announcements

### Admin
- Full system access
- User management (create, edit, deactivate)
- Module permission assignment
- System configuration
- Database administration
- Audit log access

### Super Admin
- Everything an admin can do
- Database backup and restore
- Data import and export
- Full system control

## Module Permissions

Access to each module is controlled by permissions:
- Sales, Logistics, Tenders, Project Management
- Human Resource, Stock Management
- Support Tickets, Messaging, AI
- PBX, Calendar, Settings

Admins assign permissions per user. Users only see modules they have access to.

## Operating Companies

The intranet uses SignalR for live updates:
- **Attendance**: Real-time clock in/out notifications
- **Kanban Boards**: Live card moves, creates, updates, deletes
- **Messaging**: Instant message delivery, typing indicators
- **Collaborative Docs**: Real-time collaborative editing, cursor positions

### Troubleshooting

**Page Not Loading**: Clear browser cache, try a different browser, check internet connection
**Can't Login**: Check email/password, contact admin for password reset
**Data Not Showing**: Refresh page, check filters, verify module permissions
**Real-time not updating**: Refresh page, check internet connection (SignalR will auto-reconnect)
